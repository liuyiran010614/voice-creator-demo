# 03 - 设计规范

## 一、整体风格
简洁、直观、留白充足。白底为主，百度蓝点缀。参考百度品牌视觉语言。

## 二、色板
- 主色（百度蓝）：`#4E6EF2`
- 主色深（强调 / hover）：`#2932E1`
- 主色浅（选中底 / 浅背景）：`#EBF0FF`
- 文字主色：`#1F1F1F`
- 文字次色：`#646464`
- 文字弱色 / placeholder：`#9195A3`
- 页面背景：`#F5F7FA`
- 卡片背景：`#FFFFFF`
- 分割线 / 边框：`#E8EAF0`
- 成功色：`#30C453`
- 禁用态：背景 `#C4C8D3`，文字 `#FFFFFF`

## 三、字体与字号
- 字体：系统默认无衬线（PingFang SC / Microsoft YaHei / sans-serif）。
- 页面标题：22px / 600
- 板块标题：16px / 600
- 正文 / 输入：14px / 400
- 辅助文字：12px / 400

## 四、间距与圆角
- 页面内容最大宽度：720px，居中。
- 板块间距：24px。
- 卡片内边距：20px。
- 圆角：卡片 12px，按钮 8px，输入框 8px，头像圆形。
- 元素间常用间距：8 / 12 / 16px。

## 五、组件规范
- 主按钮：百度蓝实心 `#4E6EF2`，白字，圆角 8px，hover 变 `#2932E1`；禁用态用禁用色且不可点。
- 次按钮（如"换一批"）：白底、蓝字、蓝色描边。
- 输入框：白底、`#E8EAF0` 描边，聚焦时描边变主色，圆角 8px。
- 角色卡片：白底圆角卡片，顶部圆形头像（纯色底 + emoji），下方角色名；hover 微浮起（阴影加深）。
- 音色结果卡片：白底卡片，选中时边框变主色 + 浅蓝底 `#EBF0FF`。
- 弹窗：居中卡片，半透明黑色遮罩（`rgba(0,0,0,0.45)`），圆角 12px。
- 阴影：卡片 `0 2px 8px rgba(0,0,0,0.06)`，弹窗 `0 8px 32px rgba(0,0,0,0.16)`。

## 六、交互反馈
- 按钮 hover / active 有颜色过渡（transition 0.2s）。
- 试听播放：按钮变播放态图标 + 进度条动画。
- loading：生成音色时显示转圈或骨架占位。
- 保存成功：顶部或弹窗内绿色成功提示。
